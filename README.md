# pscopy

A curses TUI tool for backing up PS1 and PS2 discs to BIN/CUE or ISO files, with automatic game identification via a scraped database from [psxdatacenter.com](https://psxdatacenter.com).

## Features

- **Curses TUI** with status display, progress bar, and scrolling log
- **Automatic disc detection** via pyudev (handles discs already in the drive)
- **Game database** — 21,000+ titles scraped from psxdatacenter.com (all regions)
- **Quick serial entry** — type just the 5-digit suffix after the first game and pscopy searches the same region
- **PS1 dumps** via `cdrdao` → BIN/CUE/TOC (raw read with L-EC error detection)
- **PS2 dumps** via `dd` → ISO (with sync/noerror and progress tracking)
- **Verified dump naming** — files tagged `[!]` for clean dumps or `[b]` for bad dumps
- **Auto-eject** after each dump

## Requirements

- Python 3.6+
- `cdrdao` (PS1 disc dumping)
- `dd` (PS2 disc dumping)
- `udevadm` (media detection)
- An optical drive that can read PS1/PS2 discs

### Python packages

```
pip install -r requirements.txt
```

## Usage

### 1. Build the game database

```bash
python scrape_db.py --verbose
```

This scrapes psxdatacenter.com and creates `games.db` (~21,000 entries across PS1/PS2, NTSC-U/PAL/NTSC-J).

### 2. Run pscopy

```bash
sudo python pscopy.py -o /path/to/output --device /dev/sr0
```

Root or appropriate permissions are needed for raw disc access.

### Options

| Flag | Default | Description |
|------|---------|-------------|
| `-o`, `--output` | *(required)* | Output directory for dumps |
| `--db` | `games.db` | Path to game database |
| `--device` | `/dev/cdrom` | Optical drive device |

### Workflow

1. Insert a disc (or start with one already in the drive)
2. Enter the game serial (e.g. `SLUS-00594`) or a custom title
3. For subsequent discs from the same region, just type the 5-digit number (e.g. `00594`)
4. Confirm the dump
5. Disc is dumped, verified, named, and ejected
6. Insert the next disc

### Output naming

```
Title (Disc 1) (US) [SLUS-00594] [!].bin
Title (Disc 1) (US) [SLUS-00594] [!].cue
Title (Disc 1) (US) [SLUS-00594] [!].toc
```

- `[!]` = verified good dump (no errors)
- `[b]` = bad dump (errors detected)

### Keyboard

- **q** — quit while waiting for a disc

## Files

| File | Description |
|------|-------------|
| `pscopy.py` | Main backup tool (curses TUI) |
| `scrape_db.py` | Database scraper for psxdatacenter.com |
| `requirements.txt` | Python dependencies |
